# 💰 Expense Tracker

A simple and responsive **Expense Tracker** web application built using **HTML**, **CSS**, and **JavaScript**. This project provides a clean interface to help users add and manage their income and expense entries while displaying the current balance.

## 🚀 Features

* ➕ Add income and expense entries
* 💰 Display current balance
* 📊 Show total income and total expenses
* 🗑️ Delete transactions
* 📱 Responsive and user-friendly interface
* ⚡ Built with pure HTML, CSS, and JavaScript

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/praveenyadav11/expense-tracker.git
```

### Navigate to the project folder

```bash
cd expense-tracker
```

### Run the project

Open the `index.html` file in your preferred web browser.

No installation or additional dependencies are required.

---

## 🎯 How to Use

1. Enter the transaction details.
2. Specify whether it is an income or an expense.
3. Add the transaction.
4. View the updated balance, income, and expense totals.
5. Delete transactions if needed.

> **Note:** This project does not save data permanently. All transaction data is cleared when the page is refreshed.
